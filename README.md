
# GOAT Project (E-Commerce)

This is an e-commerce project powered by next.js and prisma


## Tech Stack

Next, Typescript, TailwindCSS, Prisma, React Query, TRPC and much more



## Features

- Complete auth flow
- Search
- Filters
- Different categories
- Dark/Light mode
